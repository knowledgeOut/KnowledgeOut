# Hooks

커스텀 React 훅 디렉토리

## 예상 훅 구조

- **useAuth.js**: 인증 관련 훅
  - 현재 로그인 상태 확인
  - 사용자 정보 관리
  - 로그인/로그아웃 함수

- **useQuestion.js**: 질문 관련 훅
  - 질문 목록 조회
  - 질문 상세 조회
  - 질문 CRUD 작업

- **useAnswer.js**: 답변 관련 훅
  - 답변 조회
  - 답변 CRUD 작업

- **useLike.js**: 추천 관련 훅
  - 질문/답변 추천 기능

- **useSearch.js**: 검색 관련 훅
  - 태그 기반 검색

- **useSort.js**: 정렬 관련 훅
  - 질문 정렬 (추천순 등)
