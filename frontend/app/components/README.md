# Components

공통 컴포넌트 디렉토리

## 예상 컴포넌트 구조

- **Header**: 상단 네비게이션 바
- **Footer**: 하단 푸터
- **QuestionCard**: 질문 카드 컴포넌트
- **AnswerCard**: 답변 카드 컴포넌트
- **Tag**: 태그 컴포넌트
- **CategorySelect**: 카테고리 선택 컴포넌트
- **TagInput**: 태그 입력 컴포넌트
- **LikeButton**: 추천 버튼 컴포넌트
- **SearchBar**: 검색 바 컴포넌트
- **SortMenu**: 정렬 메뉴 컴포넌트
- **AuthForm**: 인증 폼 컴포넌트 (로그인/회원가입 공통)
- **QuestionForm**: 질문 작성/수정 폼 컴포넌트
- **AnswerForm**: 답변 작성/수정 폼 컴포넌트
