# 프로젝트 기능 명세서

---

## 1. 사용자 인증 및 관리

---

### 1-1. 회원 가입
사용자는 질문 및 답변 작성을 위해서 회원 가입을 해야 합니다.
- 이메일, 비밀번호(8자 이상), 닉네임 입력
- 이메일 및 닉네임 중복 검사
- 비밀번호는 BCrypt로 암호화되어 저장

### 1-2. 로그인
사용자는 로그인을 해서 인증을 할 수 있고 인가 받을 수 있습니다.
- 이메일과 비밀번호로 로그인
- Spring Security 세션 기반 인증 사용
- 로그인 성공 시 세션 쿠키(JSESSIONID) 발급

### 1-3. 로그아웃
사용자는 로그아웃을 해서 인증 인가를 철회할 수 있습니다.
- 세션 무효화 및 쿠키 삭제

### 1-4. 계정 정보
사용자는 마이페이지에서 자신의 정보를 확인할 수 있습니다.
- 이메일, 닉네임, 가입일시 등의 정보 표시

### 1-5. 회원 정보 수정
사용자는 마이페이지에서 자신의 정보를 수정할 수 있습니다.
- 닉네임 변경 가능
- 비밀번호 변경 가능 (8자 이상, 현재 비밀번호와 동일 불가)

### 1-6. 회원 탈퇴
사용자는 마이페이지에서 회원 탈퇴를 할 수 있습니다.
- Soft delete 방식으로 처리 (MemberStatus.DELETED)
- 이메일은 null로 설정, 닉네임은 고유한 값으로 변경

### 1-7. 질문 내역
사용자는 **자신이 작성한 질문**들을 확인할 수 있습니다.
- 마이페이지에서 자신이 작성한 모든 질문 목록 조회
- 생성일시 내림차순 정렬

### 1-8. 답변 내역
사용자는 **자신이 작성한 답변**들을 확인할 수 있습니다.
- 마이페이지에서 자신이 작성한 모든 답변 목록 조회
- 생성일시 내림차순 정렬
- 삭제된 답변(status=true)은 제외

### 1-9. 추천 질문 내역
사용자는 **자신이 추천한 질문**들을 확인할 수 있습니다.
- 마이페이지에서 자신이 추천한 모든 질문 목록 조회

## 2. 관리자 기능

---

### 2-1. 질문 삭제
관리자는 이미 작성되어 있는 질문을 삭제할 수 있습니다.
- ADMIN 권한이 있는 사용자만 접근 가능
- 모든 질문에 대해 삭제 권한 보유

### 2-2. 답변 삭제
관리자는 이미 작성되어 있는 답변을 삭제할 수 있습니다.
- ADMIN 권한이 있는 사용자만 접근 가능
- 모든 답변에 대해 삭제 권한 보유

### 2-3. 대시보드
관리자는 대시보드에서 질문 및 답변에 대한 통계 정보를 확인할 수 있습니다.
- 전체 질문/답변 통계
- 카테고리별 통계
- 상태별 통계 등

---

## 3. 데이터 관리

---

### 3-1. Soft Delete
삭제 작업은 데이터 무결성을 위해 Soft Delete 방식을 사용합니다.

#### 회원 탈퇴
- `Member` 엔티티의 `status` 필드를 `DELETED`로 변경
- 이메일은 null로 설정, 닉네임은 고유한 값으로 변경

#### 답변 삭제
- `Answer` 엔티티의 `status` 필드를 `true`로 변경
- 실제 데이터베이스에서 삭제되지 않음
- 조회 시 `status=false`인 답변만 표시

## 4. 질문 작성

---

### 4-1. 카테고리 설정
사용자는 자신이 할 질문의 카테고리 설정을 할 수 있습니다.
- 질문 작성 시 카테고리 선택 가능
- 카테고리는 필수 입력 항목

### 4-2. 질문 글 작성
사용자는 자신이 원하는 질문을 작성할 수 있습니다.
- 로그인한 사용자만 질문 작성 가능
- 제목, 내용, 카테고리 필수 입력
- 태그는 선택 사항 (복수 개 가능)

### 4-3. 질문 글 수정
사용자는 자신이 작성한 질문을 수정할 수 있습니다.
- 작성자만 수정 가능
- 질문 수정 페이지에서 수정

### 4-4. 질문 글 삭제
사용자는 자신이 작성한 질문을 삭제할 수 있습니다.
- 작성자만 삭제 가능

### 4-5. 태그 추가
사용자는 질문 글 작성 시 태그를 추가할 수 있습니다.
- 태그는 복수 개 가능
- 태그 이름은 문자열 배열로 전달

### 4-6. 질문 추천
사용자는 질문을 추천할 수 있습니다.
- 로그인한 사용자만 추천 가능

## 5. 답변 작성

---

### 5-1. 답변 작성
사용자는 다른 사용자의 질문에 답변을 작성할 수 있습니다.
- 질문 상세 페이지에서 답변 작성 폼 제공
- 로그인한 사용자만 답변 작성 가능
- 답변 내용은 필수 입력

### 5-2. 답변 수정
사용자는 자신의 답변을 수정할 수 있습니다.
- 질문 상세 페이지에서 본인이 작성한 답변에만 수정 버튼 표시
- 인라인 수정 방식 (별도 페이지 이동 없이 현재 페이지에서 수정)
- 수정 버튼 클릭 시 해당 답변이 편집 모드로 전환

### 5-3. 답변 삭제
사용자는 자신의 답변을 삭제할 수 있습니다.
- 질문 상세 페이지에서 본인이 작성한 답변에만 삭제 버튼 표시
- Soft delete 방식으로 처리 (데이터베이스에서 실제 삭제되지 않음)
- 삭제된 답변은 `status=true`로 설정되어 목록에서 제외됨

### 5-4. 답변 목록 조회
질문 상세 페이지에서 해당 질문에 달린 모든 답변을 확인할 수 있습니다.
- 생성일시 오름차순 정렬
- 각 답변에 작성자, 작성일시 표시
- 삭제되지 않은 답변(status=false)만 표시

### 5-5. 답변 추천
사용자는 작성된 답변을 추천할 수 있습니다.
- 로그인한 사용자만 추천 가능

## 6. 질문 게시판

---

### 6-1. 질문 작성
사용자는 게시판에서 질문 작성이 가능합니다.
- 질문 작성 버튼 클릭 시 질문 작성 페이지로 이동
- 로그인하지 않은 경우 로그인 다이얼로그 표시

### 6-2. 질문 상세 조회
사용자는 게시판에서 특정 질문을 클릭해 해당 질문의 상세 정보를 조회할 수 있습니다.
- 질문 제목, 내용, 작성자, 작성일시, 조회수, 답변 개수 표시
- 카테고리 및 태그 정보 표시
- 해당 질문에 달린 모든 답변 목록 표시 (답변 작성 폼 포함)
- 질문 추천 기능

### 6-3. 질문 검색
사용자는 검색어를 이용해서 원하는 질문을 검색할 수 있습니다.
- 제목 및 내용에서 검색어 검색
- 카테고리별 필터링 가능
- 상태별 필터링 (전체/미답변/답변완료)

### 6-4. 질문 정렬 및 필터링
사용자는 질문들을 다양한 방식으로 정렬하고 필터링할 수 있습니다.
- 기본 정렬: 생성일시 내림차순 (최신순)
- 카테고리별 필터링
- 상태별 필터링 (미답변/답변완료)
- 페이지네이션 지원

### 6-5. 질문 개수 표시
게시판에서 각 상태별 질문 개수를 실시간으로 확인할 수 있습니다.
- 전체 질문 개수
- 미답변 질문 개수
- 답변완료 질문 개수
